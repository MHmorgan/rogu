Rogu
====

Roger's weird offspring and my personal UNIX assistant.


Commands
--------

* doctor - check if everything is ok.
    * Are dotfiles installed?
* sync - install/update/fix the system.
    * Rogu self
    * Dotfiles
    * Get-targets
    * Custom scripts
* config
* boilerplate
